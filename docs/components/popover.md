# Popover 弹出框 
常用的操作按钮

## 简单用法
<ClientOnly>
<button-demo/>
</ClientOnly>

## 组合用法
<ClientOnly>
<button-group-demo></button-group-demo>
</ClientOnly>

## Popover 属性
|   属性   |   说明   |  类型  |          可选值          | 默认值 |
| :------: | :------: | :----: | :----------------------: | :----: |
| position |   位置   | string | left、right、top、bottom |  top   |
| trigger  | 激活方式 | string |       click、hover       | click  |
