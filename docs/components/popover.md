# Popover 弹出框 
点击/鼠标移入元素，弹出气泡式的卡片浮层。

## 基础用法
<ClientOnly>
<popover-demo/>
</ClientOnly>



## Popover 属性
|   属性   |   说明   |  类型  |          可选值          | 默认值 |
| :------: | :------: | :----: | :----------------------: | :----: |
| position |   位置   | string | left、right、top、bottom |  top   |
| trigger  | 激活方式 | string |       click、hover       | click  |
